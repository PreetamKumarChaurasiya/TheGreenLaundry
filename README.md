# 🌿 The Green Laundry Website

A modern and responsive website for **The Green Laundry**, a premium laundry and dry-cleaning service based in Varanasi. This project showcases services, pricing, and business information with a clean UI and user-friendly design.

---

## 📌 Features

- ✅ Responsive design (mobile & desktop friendly)
- ✅ Clean and modern UI
- ✅ Navigation bar with social links
- ✅ About section with business details
- ✅ Services and price listing pages
- ✅ Footer with contact and address
- ✅ Easy to customize

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (optional for interactivity)

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
